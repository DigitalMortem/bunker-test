# BUNKER LAST HOPE

Разбитая и упорядоченная версия игры.

## Структура

- `index.html` — разметка страниц и подключение файлов.
- `css/styles.css` — весь внешний вид игры.
- `js/config.js` — Firebase и базовые настройки онлайн-синхронизации.
- `js/data/packs.js` — паки, темы и сценарии.
- `js/data/events.js` — события раунда и юмор исходов.
- `js/data/diseases.js` — ухудшение болезней и смерти.
- `js/data/cards.js` — карточки профессий, здоровья, багажа, фобий и т.д.
- `js/app.js` — основная логика: лобби, игроки, раунды, голосование, рендер.

## Запуск

Открой `index.html` в браузере.

Если браузер или Firebase начнут ругаться на локальный запуск, запусти папку через простой локальный сервер, например в VS Code через расширение Live Server.


## Структура паков

Данные игры разделены по пакам:

- `js/packs/registry.js` — список паков и проверка готовности.
- `js/packs/classic/` — заполненный классический пак: сценарии, карточки, болезни, события и спецвозможности.
- `js/packs/apocalypse.js`, `zombie.js`, `fantasy.js`, `maniac.js`, `space.js`, `virus.js`, `winter.js`, `cyber.js`, `island.js`, `underwater.js` — пустые заготовки для будущих паков.

Сейчас запустить игру можно только с паком `Классический`. Остальные паки видны в списке, но при запуске показывают предупреждение, потому что они ещё пустые.
